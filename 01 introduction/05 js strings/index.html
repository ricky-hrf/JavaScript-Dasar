Metode dan properti string

  • length : mengembalikan panjang string
  • toUpperCase()/toLowerCase() : mengubah huruf besar/kecil
  • slice(start,end): mengambil bagian dari string berdasarkan index
  • replace(find,replace): mengganti sebagian string dengan string lain
  • split(separator): memecah string menjadi array berdasarkan pemisah tertentu
